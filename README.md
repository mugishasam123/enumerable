# DECODE MORSE CODE
## Decoding morse code with ruby

	
## Technologies
Project is created with:
* Ruby

 ## Setup
To run this project, install it locally using:
- cd Desktop
- git clone https://github.com/VuDej/decode-morse.git
- irb
- require "./decode_morse.rb"
- decode_message('.-   -... --- -..-   ..-. ..- .-.. .-..   --- ..-.   .-. ..- -... .. . ...')
